CLEAR.js, AJAX Library for HTML/OS

An "invisible" AJAX Library. Add AJAX to any link, form, or form element on any page. 

With your normal forms, elements, or links...
 <form action="doform">
 <input type="text" name="myname">
 <input type="text" name="myemail">
 <a href="page.html">Link</a>

Simply add AJAX...
 <form action="doform" AJAX>
 <input type="text" name="myname" AJAX>
 <input type="text" name="myemail" AJAX="mousedown">
 <a href="page.html" AJAX>Link</a>

Documentation

Library Features

  • Standalone
  • Fast
  • 17KB on the client
  • Uses existing links and forms
  • Tested with IE6,7, and 8, Safari, Firefox, and Chrome
  • should work with all newer javascript enabled browsers.

Client HTML Features

  • Simply Add the keyword "AJAX" to forms, form elements and links on any page.
  • AJAX with no javascript code.
  • AJAX Uploads, no Flash
  • Multiple File Uploads

Server HTML/OS Features

  • use writeHTML(element,html) in overlays just like the display tag
  • ERRORS are handled gracefully making debugging your code easy
  • Overlays no longer use goto, use endOVERLAY() or AJAXgoto('otherpage') instead
