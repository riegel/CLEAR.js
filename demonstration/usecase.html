CLEAR.js, AJAX Library for HTML/OS

Simple use case

Suppose you have a web 1.0 site that works. And you decide you want to spice it up a bit. If you can detect what is coming in at the server then you can AJAX enable it quickly.

When a request comes in you simply (at the server) determine if the variable ajaxrequest=true. If it is then you can simply deliver the javascript back to update the part that needs it. Otherwise you simply deliver the page asked for. 

<!-- Engine: Custom PERL or PHP or Python or ??? -->
<!doctype html>
<html lang="en">
 <head>
  <meta charset="utf-8">
  <title>No HTML/OS Demo</title>
  <script src="http://clearjs.org/apps/clear/clear.js"></script>
 </head>
 <body>
  <a href="http://clearjs.org">Home</a><hr>
  <form action="nohtmlos.html" method="post" AJAX>
   <input type="text" name="myname" AJAX>
   <input type="submit" name="mybutton" value="submit">
  </form>
  <hr><a href="custom.html" AJAX>Try this (AJAX)</a> | 
  <a href="custom.html">Try this (NOAJAX)</a>
 </body>
</html>
